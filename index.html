Diagnóstico Wi‑Fi

Grave um vídeo do ambiente com dados técnicos da rede Wi‑Fi na tela para facilitar a análise de sinal, canal, interferência e estabilidade.

Leia as informações da página antes de instalar. Em caso de dúvida, fale com o analista que enviou o link do app.

Finalidade de uso

Este aplicativo é indicado para diagnósticos orientados pelo analista que enviou o link do app. Ele ajuda a registrar o ambiente onde tablets, celulares ou equipamentos estão usando a rede Wi‑Fi.

Permissões usadas

  • Câmera: para gravar o ambiente.
  • Microfone: para incluir áudio no vídeo.
  • Localização: necessária no Android para obter dados técnicos da rede Wi‑Fi.
  • Notificações: quando aplicável, para avisos do diagnóstico.

Como instalar no Android

  1. Toque no botão de download acima.
  2. Abra o arquivo APK baixado.
  3. Se o Android pedir, permita instalar apps desconhecidos para o navegador ou app usado no download.
  4. Confirme a instalação e abra o aplicativo.
  5. Conceda as permissões solicitadas pelo app.
Aviso de segurança

Como este app é distribuído fora da Play Store, o Android pode exibir avisos de segurança. Instale apenas se recebeu este link do analista responsável. Em caso de dúvida, fale com o analista que enviou o link do app antes de instalar.
